# Introduction to Git

## Create a new Git Repo

```
git init
```

```
git status
```

## To add files to Git

```
git add <filename1><filename2>
```
```
git commit -m 'Add README file about Git'
```
```
git log
```

## Check git remote

```
git remote show origin
```

## Push local branch to remote

```
# git push <remote name><branch name>
git push origin master
```
